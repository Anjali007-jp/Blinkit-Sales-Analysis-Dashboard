# Blinkit-Sales-Analysis-Dashboard
This project presents an interactive Power BI dashboard built using the Blinkit sales dataset. The dashboard provides insights into sales performance, customer satisfaction, product categories, and outlet distribution through dynamic visualizations and KPIs.
📊 Key Performance Indicators (KPIs)
💰 Total Sales
📈 Average Sales
📦 Number of Items Sold
⭐ Average Customer Rating
📌 Dashboard Insights
Total Sales by Fat Content
Total Sales by Item Type
Fat Content Analysis by Outlet
Sales by Outlet Establishment Year
Sales Distribution by Outlet Size
Sales Analysis by Outlet Location
Performance Comparison by Outlet Type
🛠️ Tools & Technologies
Power BI
Microsoft Excel (Data Source)
Power Query
DAX (Data Analysis Expressions)
📈 Features
Interactive filters and slicers
Dynamic KPI cards
Drill-down visualizations
Clean and user-friendly dashboard design
Business-focused insights for decision-making
